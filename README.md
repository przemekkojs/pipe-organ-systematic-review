In this repository, additional contents for P. Kojs, M. Kopel, D. Król: _Identifying Research Trends and Key Gaps: a Systematic Review of the Organ Pipe and Wind Instrument Simulation Research_ (2026) is presented.
The repository contains:
- VOSViewer map and network files, in `VOSViewer/` directory,
- All figures, in `Figures/` directory,
- All files in connection with Scopus AI research, in `Scopus AI` directory.
